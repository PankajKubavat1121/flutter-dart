Q1.

Who developed Dart?

**Answer:**

Dart was developed by Google.
--------------------------------

Q2.

What is Flutter?

**Answer:**

Flutter is an open-source UI framework developed by Google for building cross-platform applications from a single codebase. 
--------------------------------

Q3.

Is Dart Object Oriented?

**Answer:**

Yes, Dart is a fully object-oriented programming language.
--------------------------------

Q4.

What is Dart mainly used for?

**Answer:**

Dart is mainly used for developing Flutter applications, web applications, server-side applications, and desktop applications.

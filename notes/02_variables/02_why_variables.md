Why do we need Variables?

Variables are used to store data so that we can reuse, update, and manage it easily.

Without variables, we have to write the same value multiple times.

void main() {
print("Pankaj");
print("Pankaj");
print("Pankaj");
print("Pankaj");
}

❌ If the name changes, we must update every line.

Solution Using Variables

void main() {
String name = "Pankaj";

print(name);
print(name);
print(name);
print(name);
}
name = "Rahul";
Everything updates automatically.

Advantages of Variables

1. Reusability

Write once, use many times.

2. Easy Maintenance

Only one place to update the value.

3. Better Readability

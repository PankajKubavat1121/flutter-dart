Variable Syntax

Every variable follows this syntax:

dataType variableName = value;

String name = "Pankaj";


Understanding Each Part

String name = "Pankaj";


① String

Type (Data Type)

It tells Dart what kind of value will be stored.

Example

String

means

Only text can be stored.



② name

Variable Name (Identifier)

This is the name you give to the variable.

Example

name
age
city
salary

Use meaningful names.


③ =

Assignment Operator

It assigns (stores) the value into the variable.

Example

name = "Pankaj";

Means

Store "Pankaj" inside name.



④ "Pankaj"

Value (String Literal)

This is the actual data stored inside the variable.

Examples

"Pankaj"

"Rajkot"

"Flutter"




⑤ ;

Statement Terminator

; tells Dart that the statement has ended.

Example

String name = "Pankaj";

Without ;

❌ Compile Error

What is a Variable?

A variable is a named memory location used to store data.

Simple Definition:

A variable is a container that stores a value.



Why do we use Variables?

Without Variables

print("Pankaj");
print(28);
print("Rajkot");

પછી Value બદલવી હોય તો દરેક જગ્યાએ બદલવી પડે.

With Variables

String name = "Pankaj";
int age = 28;
String city = "Rajkot";

print(name);
print(age);
print(city);

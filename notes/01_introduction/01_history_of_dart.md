# History of Dart

## What is Dart?

Dart is an open-source programming language developed by Google.

It was designed to build:

- Mobile Applications
- Web Applications
- Desktop Applications
- Server Applications

---

## Why Dart Was Created?

Before Dart, developers mainly used JavaScript for web development.

Google wanted a language that:

- Is easy to learn
- Has better performance
- Supports Object-Oriented Programming
- Can build large applications easily

So Google created Dart.

---

## Dart Timeline

### 2011

- Dart was announced by Google.

### 2013

- First stable versions were released.

### 2017

- Dart 2 development started.

### 2018

- Dart 2 officially launched.
- Strong type system added.

### 2018+

- Flutter became popular.
- Dart became Flutter's primary language.

### Today

Dart is used for:

- Flutter Apps
- Web Apps
- Backend Development
- Desktop Apps

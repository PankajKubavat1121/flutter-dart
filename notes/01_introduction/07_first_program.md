# First Dart Program

## What is the First Dart Program?

The first program that every programmer writes is called the "Hello World" program.

It is used to verify that the Dart environment is installed and working correctly.

---

## Syntax

```dart
void main() {

}
```

The `main()` function is the entry point of every Dart program.

Execution always starts from the `main()` function.

---

## First Program

```dart
void main() {
  print("Hello World");
}
```

Output

Hello World

---

## Understanding the Code

### void

The function does not return any value.

### main()

The starting point of the Dart program.

### print()

Displays output on the console.

### "Hello World"

A String value.

---

## Execution Flow

Program Starts

↓

main()

↓

print()

↓

Output

↓

Program Ends

---

## Summary

Every Dart application starts from the `main()` function.

The `print()` function is used to display output on the console.

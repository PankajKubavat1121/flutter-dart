## First Dart Program

Q1. What is the entry point of a Dart program?

Answer:
main() is the entry point of every Dart program.

Q2. What is the purpose of print()?

Answer:
The print() function is used to display output on the console.

Q3. What does void mean?

Answer:
void means the function does not return any value.

Q4. Is main() mandatory in Dart?

Answer:
Yes. Every executable Dart program starts from the main() function.


Q1. How does Dart execute a program?

Answer:
Dart compiles the source code into machine code, and then the device executes it to produce the output.

Q2. What is the role of the Dart Compiler?

Answer:
The Dart Compiler converts Dart source code into executable machine code.

Q3. Does Dart compile code before execution?

Answer:
Yes. Dart compiles the code before it is executed.

Q4. Which component converts Dart code into machine code?

Answer:
The Dart Compiler.

Q5. What are the two compilation modes in Dart?

Answer:

JIT (Just-In-Time)
AOT (Ahead-Of-Time)



Q1. What is the full form of JIT?

Answer: Just-In-Time Compilation.

Q2. What is the full form of AOT?

Answer: Ahead-Of-Time Compilation.

Q3. When is JIT used?

Answer: JIT is used during development to provide faster development and Hot Reload.

Q4. When is AOT used?

Answer: AOT is used for release builds to improve startup time and application performance.

Q5. Which compilation mode supports Hot Reload?

Answer: JIT.

Q6. Which compilation mode provides better performance?

Answer: AOT.

Q7. Which compilation mode is used when building an APK?

Answer: AOT.
